# Car Mileage Prediction System

A modern, frontend-only web application that predicts vehicle fuel efficiency using a dynamic rule-based scoring algorithm built with HTML, CSS, and Vanilla JavaScript.

(Note:Built with the help of Google Anti-gravity,and this Project is done for Academic Purposes and Demonstration)

Designed with a futuristic Glassmorphism UI,
 The project provides real-time mileage predictions, smart parameter linking, and actionable fuel-saving suggestions — all without requiring any backend or external APIs.


## Project Overview:

The Car Mileage Prediction System helps users estimate vehicle mileage based on:

- Vehicle type,
- Fuel type,
- Transmission,
- Driving style,
- Engine power,
- Cargo load,
- AC usage,
- Driving environment.

The system evaluates user selections using deterministic weighted scoring logic and instantly displays:

- Predicted mileage,
- Efficiency breakdown,
- Personalized improvement suggestions.

This project demonstrates how intelligent frontend logic can simulate real-world prediction systems without Machine Learning or server-side processing.

## How It Works:

The system starts with a baseline mileage value.

Each user selection applies:
- Positive bonuses
- Negative penalties

Example:
- Highway driving → Mileage increase
- Aggressive driving → Mileage decrease
- Heavy cargo → Efficiency reduction

The final score is processed and displayed instantly along with optimization suggestions.
